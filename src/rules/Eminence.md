# Eminence

*Wonderland of the Enigma · System Alignment Index · Year 3261*

**Eminence** measures how the machinery of Wonderland interprets you: as an **asset**, a **liability**, or an unpredictable **anomaly**. It ranges from **−20 (System Liability)** to **+20 (System Asset)**, beginning at **0**.

Eminence is **not** about likability. It is about alignment with optimization logic.

| | |
|---|---|
| Preserving a system at human cost | Eminence tends to **rise** |
| Creating inefficiency to save lives | Eminence tends to **fall** |
| Clean, optimized, measurable outcomes | **Raise** your standing |
| Instability, ambiguity, unpredictability | **Erode** it |

---

## 1 · How Eminence Shapes Advancement

Eminence never modifies EXP or raw speed. It alters **how advancement manifests**.

### Advancement Shape

| | |
|---|---|
| **High Eminence** | Clean promotions, expanded permissions, sanctioned Talents and interfaces — "approved" upgrades that make you a more efficient tool. |
| **Low Eminence** | Conditional or distorted advancements — corrupted Talents, intrusive monitoring, failsafes embedded into your growth. |

### Advancement Cost

| | |
|---|---|
| **Positive Eminence** | Fewer explicit strings; more assumed compliance. |
| **Negative Eminence** | Advancement still occurs, but **something is taken**: memories rewritten or extracted · access revoked or rerouted · identity markers and records altered or redacted. |

### World Response

| | |
|---|---|
| **High Eminence** | Systems accommodate you — doors open, protocols flex, agents cooperate. |
| **Low Eminence** | Systems adapt **against** you — locks tighten, surveillance escalates, resistance hardens. |

Progress is never fully blocked; the **form** of your progression grows more hostile, parasitic, or invasive as Eminence drops.

---

## 2 · Gaining and Losing Eminence

Eminence changes are tied to **impact, not intent**.

- Preserving optimization while allowing collateral suffering → Eminence **rises**.
- Sabotaging efficiency to preserve autonomy or life → Eminence **falls**.
- Delivering clean, measurable, stable outcomes → Eminence **rises**.
- Introducing chaos, ambiguity, or unresolved variables → Eminence **falls**.

The Curator applies shifts **sparingly and narratively**, announcing changes in-fiction:

> *"Your intervention stabilizes the planetary network, but three communities vanish from the registry. The system logs this as a success: **+3 Eminence**."*

---

## 3 · How Advancement Feels in Play

A character qualifies for **Designation 4**. The system presents options:

| | |
|---|---|
| **Full Promotion** | Gain full Rank benefits and broader permissions — but lose access to an older identity, community, or unmonitored space. |
| **Partial Promotion** | Gain specific improvements or Talents — but accept tighter oversight, embedded restraints, or standing directives. |
| **Deferred Promotion** | Remain at a lower Designation with less visibility. Avoid immediate scrutiny — but flag yourself as **anomalous**, inviting later correction. |

## 4 · Tracker Reference

Eminence is tracked on the character sheet banner (a set number, −20 … +20). State bands as read by the Indra network:

| Range | Reading |
|---|---|
| +15 … +20 | **System Asset** |
| +6 … +14 | Favored Instrument |
| −5 … +5 | Unresolved Variable |
| −14 … −6 | Flagged |
| −20 … −15 | **System Liability** |
